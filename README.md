# To run the application

### in your terminal change your directory to /client
cd client

### install npm
npm install

### to run the website
npm run dev



# Structure and Filesystem

### pages 
contains the pages to be displayed

### layout 
contains the page layouts like navabar and footer

### components 
contain the reusable componenets

### public 
contains the images and media

### index.html 
main DOM file over which the react renders

### index.css
global css, imports, font declations, etc 

### app.css
main application shell css
